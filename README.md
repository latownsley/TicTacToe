# Tic Tac Toe (Simple)

Made for an end of the year C++ Game Making Challenge, Tic Tac Toe is a simple c++ iteration of Tic Tac Toe. It is a simple, interactive, two-player Tic-Tac-Toe game written in C++. It runs in the console and allows two players to take turns entering positions on a 3Ã—3 board until a win or draw occurs.
